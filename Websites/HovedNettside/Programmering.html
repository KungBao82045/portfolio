HTML, CSS, JavaScript

HTML er den første jeg lærte før jeg begynte med CSS og JavaScript. Jeg har prøvd å lage nettside før, men jeg brukte nettsiden bare for å lage en link som sender deg til RickRoll. Hvis du ønsker å se på koden på nettsidet mitt, høyreklikk på nettsiden og trykk på "Vis sidekilde". Hvis du er på Engelsk, trykk på "View Page Source". Hvis du ønsker å se hvordan nettsiden funker, høyreklikk på nettsiden og trykk på "inspiser", eller på Engelsk, "Inspect".
